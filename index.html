Tutorial para instalação

Aviso: Esse aplicativo funciona apenas em dispositivos com sistema operacional Android.

Após baixar o aplicativo, consulte a pasta "Downloads" do seu smartphone, o arquivo baixado irá aparecer como na imagem abaixo:

Clique no arquivo "evolua-saude.apk" para fazer a instalação do aplicativo. Será exibida uma mensagem semelhante à da imagem abaixo, solicitando permissão para instalar aplicativos de fontes desconhecidas, isso ocorre, pois o aplicativo não está na playstore.

Basta conceder essa permissão.

Clique no botão "Instalar" e a instalação irá ocorrer normalmente. Seguindo esses passos o aplicativo estará pronto para uso.

Você reconhecerá o aplicativo no seu smartphone pelo ícone a seguir.
